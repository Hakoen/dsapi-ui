What is datasets.at?

datasets.at is a mirror for SmartOS datasets/images located in Germany.

How can I use this mirror?

To be able to use datasets.at as a source for datasets you need to add http://datasets.at/datasets to the sources list of imgadm and refresh the dataset list.

datasets.at syncs with the default dataset server provided by joyent so it is also possible to replace the default entry completely.

echo "http://datasets.at/datasets" > /var/db/imgadm/sources.list
imgadm update

How do I switch back to the default source?

echo "https://datasets.joyent.com/datasets" > /var/db/imgadm/sources.list
imgadm update

What is the «Builder»?

The builder is a short (work-in-progress) formula to generate a JSON file for use with vmadm on a SmartOS host.

The formular is able to dynamically add nics, disks (or filesystems) and metadata to the JSON file. It also tries to simplify adding metadata by providing a list of supported metadata keys for the selected dataset/image.
